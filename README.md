# ChainReaction

Simple implementation of Chain Reaction Game in **Python 3**

This is a 2 player game and screen and grid can be customized in the code

     # Screen Size
     display_height = <value> 
     display_width = <value>
     
     # Grid
     rows = <value>
     columns = <value>

## Prerequisite

 - Python 3

## How to play

Type this command on console:

    python play.py
